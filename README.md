¡Hola buenas! Esta firma la podeis modificar a vuestro gusto, esta diseñada para que sea toda interactiva, es decir que si le das una imagen se te abrirá lo que tu quieras enlazar, se puede usar en gmail o outlook, instrucciones de uso:
  
  -Abrir Gmail.
  -Darle a la ruletita de ajustes que esta arriba a la derecha.
  -Cuando se despliegue le das a "Ver todos los ajustes"
  -Al darle click se te abriran todos los ajustes, baja hasta el final, y encontrarás un apartado que pone Firma.
  -Le das a "Crear".
  -Pones un nombre, por ejemplo "Firma nº1".
  -Copias todo el codigo HTML y lo pegas en la zona que hay para escribir.
  -Ahora bajas hacia abajo del todo de la página y le das a "Guardar cambios".
  -Ahora le das a redactar un nuevo correo, abajo en las opciones verás un icono de un boli, le das, y eliges la firma que acabamos de crear.

¡Espero que os haya servido de ayuda!
